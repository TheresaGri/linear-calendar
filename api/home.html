Linear calendar API

endpoints :

  • api/events
  • api/events/1
  • api/events?from=2023-03-01 (only events on or later than this date)
  • api/events?from=now (only future events)
  • api/events?from=2023-03-01&to=2023-03-31
  • api/events POST
  • api/events/1 DELETE
  • api/events/1 PUT
  • api/events/1 PATCH
  • api/categories
  • api/categories/1
  • api/categories POST
  • api/categories/1 DELETE
  • api/categories/1 PUT
  • api/categories/1 PATCH
